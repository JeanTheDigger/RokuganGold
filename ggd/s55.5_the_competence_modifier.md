## 55.5 The Competence Modifier

**STATUS:** LOCKED

What makes School and skills matter. Every ActionID maps to a primary skill and optional secondary (half value) via action_skill_map.json. Skill rank → modifier via competence_table.json: rank 0 = −20, rank 1 = −10, rank 2 = −5, rank 3 = 0, rank 4 = +5, rank 5 = +10, rank 6 = +15, rank 7+ = +20. SchoolType acts as a hard gate in Phase 3 for exclusive actions (Shugenja spells, lord levy, Bushi Katas, Monk Kiho).
