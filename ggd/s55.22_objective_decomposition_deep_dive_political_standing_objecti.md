## 55.22 Objective Decomposition Deep-Dive — Political Standing Objectives

**STATUS:** LOCKED

Full decomposition trees, edge cases, and worked examples for every Political Standing Objective. These are directional objectives (never completed) that guide NPC behavior when no primary objective is active. Each decomposition must answer: “What is the most productive thing I can do RIGHT NOW to advance this direction, given where I am and what I know?”

### 55.22.1 "Expand my clan’s territorial influence"

**Nature:** Aggressive, outward-facing. The NPC looks at borders and sees opportunity. This is the most war-adjacent standing objective — it is the standing objective most likely to trigger the War Justification system (Section 53).

**Who gets this:** Clan Champions, Family Daimyo, Provincial Daimyo with Kyōryōku or Seigyo Shourido. Weighted heavily for aggressive, ambitious leaders.

#### 1.1 Decomposition Tree

#### 1.2 Edge Cases

**EDGE CASE:** NPC is a Clan Champion with no weak neighbors: All borders are strong. Decomposition falls through to diplomatic expansion — build alliances and political leverage at court to create future opportunity. The objective is directional, never complete, so “no immediate opportunity” is not a failure.

**EDGE CASE:** NPC is a low-rank courtier with this standing objective: They can’t levy troops or evaluate borders. Decomposition skips all lord-tier branches and goes straight to court behavior: build relationships with neighboring clans, advocate for territorial claims at court, gather intelligence on weak neighbors to report to their lord.

**EDGE CASE:** NPC’s clan is currently at war: The standing objective is already being served by the war. Decomposition should detect active War Status and defer to the primary objective (which the lord would have assigned for the war). If no primary objective exists, the standing objective generates: SUPPORT_WAR_EFFORT — which decomposes to ACQUIRE_RESOURCE, LEVY_TROOPS, or ATTEND_COURT to advocate for continued military commitment.

**EDGE CASE:** Jin personality with this objective: Per Section 53 (War Justification), Jin prefers diplomatic expansion and only justifies military action if non-military options are exhausted. The decomposition handles this naturally because Jin’s personality filter blocks RAID_HARVEST and Forbidden Tactics, and their lean pushes them away from military actions. They’ll pursue the diplomatic path even when a military one is available.

### 55.22.2 "Maintain the balance of power — no single clan dominates"

**Nature:** Reactive, stabilizing. The NPC watches for anyone growing too powerful and acts to check them. This is the diplomat’s standing objective — it requires awareness of the broader political landscape.

**Who gets this:** Courtiers with Gi or Dōsatsu virtues. Imperial family members. Crane and Scorpion characters who serve the balance.

#### 2.1 Decomposition Tree

func _decompose_maintain_balance(obj: Objective, ctx: ContextSnapshot) -> ImmediateNeed:

var need = ImmediateNeed.new()

# ═══ STEP 1: Assess the landscape ═══

# Do I know who's currently strongest?

var clan_strengths = ctx.known_clan_strengths # NEW field from Gap 7

if clan_strengths.is_empty():

# I don't know the landscape. Gather intelligence.

need.need_type = NeedType.ASSESS_POWER_BALANCE

need.priority = 2

return need

# ═══ STEP 2: Identify the dominant clan ═══

var sorted = clan_strengths.keys()

sorted.sort_custom(func(a, b): return clan_strengths[a] > clan_strengths[b])

var dominant_clan = sorted[0]

var second_clan = sorted[1] if sorted.size() > 1 else null

# Is there a significant gap? (dominant is 20%+ ahead of second)

var gap = clan_strengths[dominant_clan] - clan_strengths.get(second_clan, 0)

var threshold = clan_strengths[dominant_clan] * 0.2

if gap < threshold:

# No single clan dominates. Balance is maintained.

# Default to maintenance: build broad relationships, stay informed.

need.need_type = NeedType.GATHER_INTELLIGENCE

need.priority = 1

return need

# ═══ STEP 3: Someone is dominant. Check them. ═══

# Am I the dominant clan? (shouldn't have this objective, but handle it)

if dominant_clan == CharacterDB.get(ctx.npc_id).clan_id:

# I'm the dominant one. This objective tells me to... restrain myself?

# Rare edge case. Default to maintenance behavior.

need.need_type = NeedType.REST

need.priority = 1

return need

# ═══ STEP 4: Another clan is dominant. Act against them. ═══

match ctx.context_flag:

ContextFlag.AT_COURT:

# At court: coalition building against the dominant clan

# Find representatives of weaker clans to build alliance

var weaker_clans = sorted.slice(1) # everyone except dominant

var coalition_contacts = _find_contacts_from_clans(weaker_clans, ctx)

if coalition_contacts.is_empty():

need.need_type = NeedType.IDENTIFY_CONTACT

need.target_clan_id = weaker_clans[0]

need.priority = 2

return need

# Have contacts from weaker clans. What's our relationship?

var best = _pick_best_contact(coalition_contacts, ctx.dispositions)

if ctx.dispositions.get(best.id, 0) < 31:

need.need_type = NeedType.RAISE_DISPOSITION

need.target_npc_id = best.id

need.threshold = 31; need.threshold_type = "disposition"

need.priority = 2

return need

# Friends with weaker clans. Now damage the dominant clan's alliances.

var dominant_contacts = _find_contacts_from_clan(dominant_clan, ctx)

if dominant_contacts.size() > 0:

need.need_type = NeedType.DAMAGE_RELATIONSHIP

need.target_npc_id = dominant_contacts[0].id

need.priority = 2

return need

# No dominant clan contacts to undermine directly.

# Move topic positions against the dominant clan at court.

need.need_type = NeedType.MOVE_TOPIC_POSITION

need.priority = 2

return need

ContextFlag.AT_OWN_HOLDINGS:

# Write letters building the anti-dominant coalition

need.need_type = NeedType.SEND_LETTER

need.priority = 1

return need

_:

need.need_type = NeedType.ATTEND_COURT

need.priority = 1

return need

# - Army sizes observed at court/through intel: weight 5

# - Active wars won/lost (from topic system): weight 2

# - Court influence (Status of attending delegates): weight 2

#### 2.3 Edge Cases

**EDGE CASE:** Two clans are equally dominant: The NPC doesn’t need to act as aggressively. They should build relationships with BOTH weaker clans and position themselves as the balancing vote at court. Decomposition handles this: the gap calculation shows no single dominant clan, so it defaults to broad relationship-building.

**EDGE CASE:** The NPC’s own clan IS the dominant one: Rare but possible if the objective was generated before the clan became dominant. The NPC should either restrain their own clan’s aggression (advocate peace at court, oppose expansion-minded leaders) or the standing objective should change via a major life event trigger (Section 18.6). For now, decomposition defaults to REST.

**EDGE CASE:** No intelligence available: First firing always returns ASSESS_POWER_BALANCE / GATHER_INTELLIGENCE. The NPC spends their first sessions learning the landscape before acting.

**EDGE CASE:** At Winter Court with the Emperor present: This is THE venue for balance-of-power politics. The NPC should prioritize building a coalition and influencing the Emperor against the dominant clan. Urgency bonus: +10 for “at Winter Court” when pursuing this objective.

#### 2.4 Worked Example

**NPC:** Kitsuki Yasu (Gi/Dōsatsu, Dragon Courtier, Rank 3). Standing obj: “Maintain balance of power.” At Crane Champion court. Known clan strengths: Lion estimated strongest (army size 25+ PU observed), Crane second, Crab third. Lion is 30% ahead of Crane.

**Phase 2:** Step 1: clan_strengths exists. Step 2: Lion dominant, gap exceeds 20%. Step 3: not Lion clan. Step 4: at court. Find weaker clan contacts — Crane contacts present (we’re at Crane court). Disposition toward best Crane contact: +22 (Acquaintance). Below +31 Friend threshold. Returns RAISE_DISPOSITION targeting the Crane contact.

**Multi-day arc:** Days 1–3: Build Crane friendship (CHARM, LISTEN_REFLECT, DELIVER_GIFT). Day 4: Disposition crosses +31. Decomposition advances: now seeks to damage Lion alliances. Are there Lion contacts at this court? If yes → DAMAGE_RELATIONSHIP (through GOSSIP, DISCLOSE, or finding dirt via ACQUIRE_LEVERAGE). If no → MOVE_TOPIC_POSITION against Lion interests on the court’s active topics. Yasu becomes the quiet orchestrator of an anti-Lion coalition — exactly the Dōsatsu/Gi archetype.

### 55.22.3 "Advance my family’s standing within the clan hierarchy"

**Nature:** Ambitious, inward-facing. The NPC wants their family to rise within their own clan. This is about Glory, visibility, and building relationships with decision-makers above them.

**Who gets this:** Junior courtiers, minor family members, ambitious provincial lords. Seigyo and Ketsui Shourido characters.

#### 3.1 Decomposition Tree

func _decompose_advance_family(obj: Objective, ctx: ContextSnapshot) -> ImmediateNeed:

var need = ImmediateNeed.new()

var my_clan = CharacterDB.get(ctx.npc_id).clan_id

# ═══ STEP 1: Who are the decision-makers in my clan? ═══

# The people who control promotions and appointments:

# Clan Champion, Family Daimyo, high-Status courtiers

var decision_makers = CharacterDB.get_by_clan_and_min_status(my_clan, 5)

var known_dm = decision_makers.filter(

func(id): return id in ctx.known_contacts.get(my_clan, []))

if known_dm.is_empty():

# I don't know any decision-makers. Network upward.

need.need_type = NeedType.IDENTIFY_CONTACT

need.target_clan_id = my_clan # within my own clan, seeking higher-ups

need.priority = 2

return need

# ═══ STEP 2: Context-dependent action ═══

match ctx.context_flag:

ContextFlag.AT_COURT:

# This is where standing is built. Two parallel tracks:

# Track A: Build disposition with decision-makers

# Track B: Seek Glory through public actions

# Which track? Depends on current disposition and Glory

var present_dm = known_dm.filter(func(id): return id in ctx.characters_present)

if present_dm.size() > 0:

var best_dm = _pick_best_contact(present_dm, ctx.dispositions)

var disp = ctx.dispositions.get(best_dm.id, 0)

if disp < 31:

# Not yet Friend with any decision-maker. Build rapport first.

need.need_type = NeedType.RAISE_DISPOSITION

need.target_npc_id = best_dm.id

need.threshold = 31; need.threshold_type = "disposition"

need.priority = 2

return need

# Already Friend+. Seek Glory to make yourself visible.

need.need_type = NeedType.SEEK_GLORY

need.priority = 2

return need

else:

# Decision-makers not at this court. Seek Glory anyway —

# word will travel through the topic system.

need.need_type = NeedType.SEEK_GLORY

need.priority = 1

return need

ContextFlag.AT_OWN_HOLDINGS:

# Between courts: write letters to decision-makers,

# train skills to perform better at next court,

# or manage territory well (a well-run province reflects on the family).

if ctx.is_lord:

# Triage provinces — a crisis in my territory hurts family standing

var ranked = _triage_provinces(ctx)

if ranked.size() > 0 and ranked[0]._triage_score > 50:

return _decompose_secure_territory(obj, ctx)

# Write letter to highest decision-maker I know

var best_dm = _pick_highest_status(known_dm)

need.need_type = NeedType.SEND_LETTER

need.target_npc_id = best_dm.id

need.priority = 1

return need

_:

# Get to a court where decision-makers will be present

need.need_type = NeedType.ATTEND_COURT

need.priority = 1

return need

#### 3.2 Edge Cases

**EDGE CASE:** NPC IS the decision-maker (Family Daimyo): They can’t advance their family by impressing themselves. Decomposition shifts to: advance the family’s reputation externally — SEEK_GLORY at higher-tier courts (Clan Champion court, Winter Court) to build the family’s name beyond the clan. Also: ensure the family’s provinces are well-run (no crises, good stability) as this reflects on their leadership.

**EDGE CASE:** NPC’s family is already the most prominent: The objective is still directional — maintain and extend. Decomposition shifts from “climb” to “defend position.” Watch for rival families gaining influence (GATHER_INTELLIGENCE), maintain strong disposition with the Clan Champion, and SEEK_GLORY to stay visible.

**EDGE CASE:** NPC is a bushi, not a courtier: They advance their family through military excellence rather than court performance. SEEK_GLORY decomposes to duel victories and battlefield achievements rather than PUBLIC_PERFORMANCE. The Competence Modifier handles this automatically — Iaijutsu scores higher than Performance for a bushi.

**EDGE CASE:** No courts scheduled and NPC is at holdings: The NPC writes letters (maintain contact with decision-makers), trains (prepare for next court), and manages territory well. Not exciting, but realistic. An ambitious samurai between courts keeps the plates spinning.

#### 3.3 Worked Example

**NPC:** Shiba Tsukune (Rei/Kanpeki, Phoenix Courtier, Rank 2). Standing obj: “Advance family standing.” At Phoenix Champion court. Knows the Family Daimyo (disposition +25, Acquaintance). Knows two high-Status Phoenix courtiers (disp +15, +8). No primary objective active.

**Phase 2:** Step 1: decision-makers found (Family Daimyo + 2 courtiers). Step 2: at court. Present decision-makers? Family Daimyo is present. Disposition +25 < +31. Returns RAISE_DISPOSITION targeting Family Daimyo.

**Phase 5:** CHARM: ObjAlign 90 + Rei +8 Kanpeki +5 = +13 (cap at 15? no, 13) + Competence (Etiquette, say 4 = +5) = 108. DELIVER_GIFT: 85 + +15 + +5 = 105. LISTEN_REFLECT: 85 + +13 + (Investigation 2 = −5) = 93. CHARM wins.

**Multi-day arc:** Days 1–3: CHARM, DELIVER_GIFT, LISTEN_REFLECT targeting Family Daimyo. Disposition climbs from +25 to ~+32 (Friend). Day 4: Decomposition advances — now Friend with decision-maker. SEEK_GLORY fires. Tsukune switches to PUBLIC_PERFORMANCE (Rei +12, Kanpeki +12 = +15 cap, high ObjAlign). She performs a tea ceremony for the court, gaining Glory and impressing the Family Daimyo who is watching. The engine naturally creates the “young courtier building her reputation under the eye of her superior” narrative.

### 55.22.4 "Undermine [Clan X]’s influence across the Empire"

**Nature:** Hostile, targeted. The NPC has a specific clan they want to weaken. This is the Scorpion archetype objective — shadow operations, broken alliances, exposed secrets.

**Who gets this:** Seigyo and Dōsatsu characters. Scorpion characters heavily. Also characters with Blood Enemy disposition toward the target clan.

#### 4.1 Decomposition Tree

func _decompose_undermine_clan(obj: Objective, ctx: ContextSnapshot) -> ImmediateNeed:

var need = ImmediateNeed.new()

var target_clan = obj.target_clan_id

# ═══ STEP 1: What do I know about the target clan? ═══

var target_contacts = ctx.known_contacts.get(target_clan, [])

var held_leverage = SecretSystem.get_held_secrets_by_clan(ctx.npc_id, target_clan)

# ═══ STEP 2: Context-driven strategy ═══

match ctx.context_flag:

ContextFlag.AT_COURT:

# AT COURT: the primary battleground for undermining

# Phase A: Find targets (if I don't know anyone from Clan X)

var present_targets = target_contacts.filter(

func(id): return id in ctx.characters_present)

if present_targets.is_empty() and target_contacts.is_empty():

need.need_type = NeedType.IDENTIFY_CONTACT

need.target_clan_id = target_clan

need.priority = 2

return need

# Phase B: Do I have leverage to deploy?

if held_leverage.size() > 0:

# I have dirt. Deploy it.

var best_secret = _pick_highest_severity_secret(held_leverage)

if _is_public_exposure_optimal(best_secret, ctx):

# Public exposure at court = maximum damage

need.need_type = NeedType.DAMAGE_RELATIONSHIP

need.target_npc_id = best_secret.subject_id

need.priority = 3

return need

else:

# Private deployment (blackmail, private reveal)

need.need_type = NeedType.DAMAGE_RELATIONSHIP

need.target_npc_id = best_secret.subject_id

need.priority = 2

return need

# Phase C: No leverage yet. Two paths:

# Path C1: Acquire leverage (covert)

# Path C2: Damage relationships directly (court actions)

# Personality determines which scores higher in Phase 5

if present_targets.size() > 0:

var best_target = _pick_highest_status(present_targets)

need.need_type = NeedType.ACQUIRE_LEVERAGE

need.target_npc_id = best_target.id

need.priority = 2

return need

else:

# Target clan not present at court. Damage their reputation

# indirectly through other attendees.

need.need_type = NeedType.DAMAGE_RELATIONSHIP

need.target_clan_id = target_clan

need.priority = 2

return need

ContextFlag.AT_OWN_HOLDINGS:

# At holdings: intelligence gathering via letters and agents

if held_leverage.is_empty():

need.need_type = NeedType.ACQUIRE_LEVERAGE

need.target_clan_id = target_clan

need.priority = 2

return need

# Have leverage but not at court to deploy it. Attend one.

need.need_type = NeedType.ATTEND_COURT

need.priority = 2

return need

_:

need.need_type = NeedType.ATTEND_COURT

need.priority = 1

return need

#### 4.2 Edge Cases

**EDGE CASE:** Gi or Makoto NPC with this objective: Personality filter blocks GOSSIP, FABRICATE_SECRET, and false DISCLOSE. They can still undermine through PUBLIC_DEBATE (arguing against the target clan’s positions), MOVE_TOPIC_POSITION (shifting court opinion), and EXPOSE_SECRET_PUBLIC (only with genuine, proven secrets). The honorable underminer is slower but uses legitimate tools.

**EDGE CASE:** Target clan is an ally of the NPC’s own clan: Complicated. The NPC’s personal standing objective conflicts with their clan’s political reality. If assigned a primary objective that contradicts this (e.g., “strengthen alliance with Clan X”), the primary objective takes precedence per the goal resolution priority cascade. The standing objective only fires when no primary is active.

**EDGE CASE:** Target clan’s representatives never attend courts the NPC can access: The NPC must work through intermediaries. DAMAGE_RELATIONSHIP targeting the target clan’s allies — weaken their alliance network rather than attacking them directly. Or INTERCEPT_LETTER to learn their plans and share them with rivals.

**EDGE CASE:** The NPC discovers their own clan is doing business with the target clan: They face a choice: undermine the trade (harming their own clan’s economy) or accept the trade and focus undermining on other fronts. Personality drives this: Chugi defers to clan leadership, Seigyo compartmentalizes, Gi raises the issue publicly.

**Phase 5:** BRIBE_FOR_INFO: 90 + (hostile, Enemy disp toward Lion contacts) +10 + Seigyo +10 + Competence (Temptation 4 = +5) = 115. EAVESDROP: 75 + 0 + Seigyo +12 + (Stealth 3 = 0) = 87. PROBE: 40 + 0 + Seigyo +12 + (Courtier 4 = +5) = 57. BRIBE wins at 115.

**Multi-day arc:** Day 1: BRIBE_FOR_INFO on Ikoma Shu’s servants. Success: acquires a Tier 3 secret. Day 2: held_leverage is no longer empty. Decomposition shifts to Phase B: deploy leverage. Is public exposure optimal? At Crane court with many witnesses → yes. Returns DAMAGE_RELATIONSHIP. Phase 5 scores EXPOSE_SECRET_PUBLIC highest (ObjAlign 90 for DAMAGE_RELATIONSHIP + secret severity bonus). Day 3: Taka exposes Ikoma Shu’s secret before the entire court. Shu’s reputation is devastated. Lion influence at this court collapses. The Scorpion does what the Scorpion does.

### 55.22.5 "Strengthen the Imperial institution"

**Nature:** Pro-establishment, stabilizing. The NPC believes in the Emperor and the Celestial Order. They work to ensure Imperial authority is respected and the institutions function.

**Who gets this:** Chugi and Meiyo characters. Imperial family members. Crane characters (historically the institution’s staunchest supporters).

#### 5.1 Decomposition Tree

func _decompose_strengthen_imperial(obj: Objective, ctx: ContextSnapshot) -> ImmediateNeed:

var need = ImmediateNeed.new()

# ═══ STEP 1: Am I near Imperial officials? ═══

var imperial_contacts = ctx.known_contacts.get("imperial", [])

match ctx.context_flag:

ContextFlag.AT_COURT:

# Is this Winter Court with the Emperor?

var court = CourtSystem.get(ctx.current_court_id)

if court.emperor_present:

# HIGHEST PRIORITY VENUE. Build disposition with Emperor.

var emperor_id = CharacterDB.get_emperor_id()

var disp = ctx.dispositions.get(emperor_id, 0)

if disp < 31:

need.need_type = NeedType.RAISE_DISPOSITION

need.target_npc_id = emperor_id

need.threshold = 31; need.threshold_type = "disposition"

need.priority = 3

return need

# Friend+ with Emperor. Advocate pro-Imperial positions.

need.need_type = NeedType.MOVE_TOPIC_POSITION

need.priority = 3

return need

# Not Winter Court. Build relationships with Imperial officials present.

var present_imperial = imperial_contacts.filter(

func(id): return id in ctx.characters_present)

if present_imperial.is_empty():

# No Imperial contacts here. Identify them.

need.need_type = NeedType.IDENTIFY_CONTACT

need.target_clan_id = "imperial"

need.priority = 2

return need

var best = _pick_best_contact(present_imperial, ctx.dispositions)

#### 5.2 Edge Cases

**EDGE CASE:** Clan is in rebellion against the Emperor: This creates an existential conflict with the standing objective. The NPC with this objective would resist their own clan’s rebellion — potentially switching sides, refusing orders, or working to end the rebellion from within. Chugi characters face the deepest conflict: duty to lord vs. duty to Emperor. The engine resolves this through the personality filter: if the lord’s orders directly contradict Imperial authority, a Chugi NPC with this standing objective may refuse (the objective is the “higher duty”).

**EDGE CASE:** NPC is low-rank with no access to Imperial circles: They strengthen the institution from below: obey Imperial edicts, enforce Imperial law in their province (if lord-tier), speak in favor of Imperial authority at lower courts, and write letters to Imperial contacts requesting instruction. The decomposition handles this naturally — IDENTIFY_CONTACT fires until they build the necessary network.

**EDGE CASE:** The Emperor is weak or corrupt (scenario-specific): The objective still fires. The NPC supports the INSTITUTION, not necessarily the individual. They may work to improve the institution from within or support a legitimate succession. This is a deep narrative scenario that the engine supports through MOVE_TOPIC_POSITION and RAISE_DISPOSITION mechanics.

#### 5.3 Worked Example

**NPC:** Otomo Yorihisa (Chugi/Chishiki, Imperial Courtier, Rank 3). Standing obj: “Strengthen Imperial institution.” At Winter Court. Emperor present. Disposition toward Emperor: +45 (Friend).

**Phase 2:** At Winter Court, Emperor present. Disposition +45 already Friend. Returns MOVE_TOPIC_POSITION (advocate pro-Imperial positions on the three active crises).

**Phase 5:** PERSUADE scores highest for position-moving (ObjAlign 95). Yorihisa argues that the Imperial response to the Shadowlands crisis should be strengthened — more authority to the Emperor’s appointed general. Day after day, he moves attendees toward supporting an Imperial Edict that concentrates military authority. This is how institutions are strengthened: through sustained political advocacy at the highest court.

### 55.22.6 "Accumulate political leverage — favors, secrets, and alliances held in reserve"

**Nature:** Patient, acquisitive. The NPC doesn’t want to USE power right now — they want to ACCUMULATE it for future use. This is the spymaster objective.

**Who gets this:** Seigyo and Chishiki characters heavily. Scorpion characters. Also careful Dragon and Crane strategists.

#### 6.1 Decomposition Tree

#### 6.2 The Three-Track Balancing Logic

The decomposition cycles between secrets, favors, and alliances by always pursuing the weakest track. This prevents the NPC from over-investing in one type of leverage:

**Low secrets + adequate favors + adequate alliances:** Focus on covert acquisition. The engine sends the NPC to BRIBE, EAVESDROP, etc.

**Adequate secrets + low favors + adequate alliances:** Focus on OFFER_FAVOR. The engine sends the NPC to help others, creating debts.

**Adequate secrets + adequate favors + low alliances:** Focus on RAISE_DISPOSITION. The engine sends the NPC to build new friendships.

**All adequate:** Cycle between all three opportunistically. The engine defaults to whichever track has the best immediate opportunity (highest-value target present, etc).

#### 6.3 Edge Cases

**EDGE CASE:** NPC’s personality blocks covert actions (Gi, Makoto, Rei): Track A (secrets) is largely closed. The NPC accumulates leverage through favors and alliances only. They are a less effective spymaster but a more trusted ally — their leverage comes from relationships, not blackmail. The engine handles this naturally through the personality filter: covert actions are removed, so OFFER_FAVOR and RAISE_DISPOSITION dominate.

**EDGE CASE:** NPC holds a devastating secret about an ally: The objective says accumulate, not deploy. The NPC holds the secret in reserve. The decomposition never returns “deploy leverage” because the standing objective is about HAVING power, not using it. Only a primary objective (assigned by a lord) would trigger deployment. This distinction matters: the NPC is a strategic hoarder, not an indiscriminate saboteur.

**EDGE CASE:** Court is empty of valuable targets: The NPC defaults to Track C (broad relationship building) at lower courts, or writes letters from holdings. Even without high-value targets, maintaining and expanding the social network IS leverage. The engine scores WRITE_LETTER and CHARM for their long-term network value.

#### 6.4 Worked Example

**NPC:** Shosuro Hametsu (Makoto/Seigyo, Scorpion Courtier, Rank 4). Standing obj: “Accumulate political leverage.” At Crane Champion court. Holds 2 secrets (Tier 4, minor). Holds 1 favor (Moderate). Has 3 Friend+ relationships. Assessment: secrets are weakest track (low-tier secrets, need better ones).

**Wait — Makoto blocks all covert actions?** Yes. Hametsu is a Makoto Scorpion. This is intentional and creates interesting gameplay. He cannot spy, bribe, or fabricate. His leverage accumulation comes entirely through favors and alliances. He’s the “honest Scorpion” — a rare and deeply interesting character type in L5R.

**Phase 2:** Three-track assessment: secrets=2 (weakest), favors=1, alliances=3. Weakest = secrets. Returns ACQUIRE_LEVERAGE.

**Phase 4:** Makoto blocks: BRIBE_FOR_INFO, EAVESDROP, INTERCEPT_LETTER, SEARCH_QUARTERS, FABRICATE_SECRET. All covert actions removed. Remaining options for ACQUIRE_LEVERAGE: PROBE (40 alignment), READ_CHARACTER (35). These are legitimate intelligence-gathering.

**Phase 5:** PROBE: 40 + Seigyo +12 + (Courtier 5 = +10) = 62. READ_CHARACTER: 35 + Seigyo +12 + (Investigation 3 = 0) = 47. PROBE wins. Hametsu asks careful questions of a Crane lord, learning their position and possibly uncovering a weakness through legitimate conversation. It’s slower than bribery, but it’s all his integrity allows.

**If Hametsu were Chugi/Seigyo instead:** All covert actions would be available (Chugi permits when lord assigns). BRIBE_FOR_INFO scores 90 + Seigyo +10 + (Temptation skill) = much higher. The engine produces a completely different Scorpion based solely on the Bushido virtue.
