## 29.15 Courtier School Implementation Framework — LOCKED

This section documents how courtier school techniques translate into the persistent world’s existing mechanical systems. The goal is to implement courtier schools using the framework already designed — Disposition System (Section 12.2), Topic Position System (Section 15.5), Court Action Menu (Section 15.4), Favor System (Section 12.10), Blackmail System (Section 12.9), Secret System (Section 12.8), Personality System (Section 19), NPC Decision Engine (Section 55), and the Named Character system (Section 22). Where the tabletop source material uses concepts that do not exist in the persistent world (XP costs, per-session limits, Devotion Ally Advantages), this section specifies the translation.

### 29.15.1 Global Translation Rules

**Per-session limits:** Any technique that grants a limited-use ability “per session” refreshes once per real-life week (OOC). The persistent world has no session boundaries. Weekly refresh provides a fixed server-side cooldown.

**Devotion Ally translation:** The tabletop Devotion Ally Advantage (1–4 points) does not exist as a separate system. “Gain target as X-point Devotion Ally” translates to a permanent historical disposition modifier in Section 12.2: Devotion 1 = +20, Devotion 2 = +35, Devotion 3 = +50, Devotion 4 = +65. These modifiers do not stack with repeated applications of the same technique on the same target.

**XP cost reductions:** Tabletop techniques that reduce XP costs for Advantages or skills are not implemented. The persistent world does not use XP as a player-facing currency.

**Temporary Advantages:** The Advantage entries on character sheets support a duration field: either permanent or a timestamp for expiry. Techniques that grant or suppress Advantages use timed entries that the system checks and clears automatically.

**Honor and Glory Scale Convention:** See Section 4.6 for the authoritative definition. Summary: 1 point = 0.1 on the scale, 1 rank = 1.0. “Lose 2 points” = −0.2. “Lose 1 rank” = −1.0.

**Lying and Deceptive Intent:** Several courtier techniques trigger against characters who are “lying.” Lying is defined as: the acting character knowingly communicates false information. A character who repeats a fabricated secret (Section 12.8) they believe to be true is not lying — their intent is honest even if the information is false. The system determines lying by checking the actor’s knowledge state and chosen skill emphasis: Sincerity (Deceit) or any action with deceptive intent constitutes lying. Sincerity (Honesty) used on information the actor believes to be true does not, regardless of whether the information is objectively false.

**New Action — DISCERN_NEED:** Several courtier schools (Yasuki R1, Doji R1) grant the ability to identify what a target currently wants. This is a distinct action from READ_CHARACTER (Category 5, Section 15.4), which reveals personality and temperament. DISCERN_NEED queries the target’s current objective stack from the NPC Decision Engine — their active resource shortages, political goals, and court session priorities. Cost: 1 AP. The skill pairing varies by school. Defense: Etiquette/Awareness (the target concealing their desires).

### 29.15.2 Crab Clan — Yasuki Courtier

**Rank 1 — The Way of the Carp:** Three effects. (1a) Free Raise on Commerce: passive modifier. Roll engine checks school field; if yasuki_courtier, add +1 Free Raise to all Commerce rolls. Same pattern as other school-specific skill bonuses. (1b) No Honor/Glory loss for public commerce: boolean flag commerce_honor_exempt on the character sheet. When the court Honor event table fires for “samurai conducted commerce publicly,” check flag and skip penalty. Multiple schools share this flag (Yoritomo Courtier R1, Ide Trader). (1c) Contested Commerce/Perception vs Etiquette/Awareness to discern what someone wants: uses the DISCERN_NEED action (see 29.15.1) with Commerce/Perception as the skill pairing instead of the Doji’s Courtier/Awareness. Same output — target’s highest-priority current objective or resource shortage.

**Rank 2 — Do As We Say:** Re-roll failed Sincerity or Intimidation rolls. Charges: School Rank per OOC week. Generic re-roll resource system: reroll_charges field with parameters (eligible skills, charge count, refresh rate). When a Sincerity or Intimidation roll fails, system checks remaining charges; if available, re-roll and decrement. NPC Decision Engine heuristic: spend charges when the target’s disposition value makes the interaction high-stakes. This re-roll resource system is generic and reusable across multiple schools.

**Rank 3 — Treasures of the Carp:** Commerce/Awareness roll (TN 20) to locate rare or useful items for others. Connects to the Commerce System (Section 51) item availability tables. Success treats the current settlement as one tier higher for item access. Each Raise increases quality tier of the found item (Normal → Fine → Exceptional per Section 49). Critical restriction: items found through this technique can only be delivered to others, not kept for personal use. On delivery, a supply ledger entry is created: supplied_item_to: {target_id, timestamp}. This ledger is read by Rank 5 and by the Yasuki Extortionist path.

**Rank 4 — Wiles of the Carp:** Passive defensive modifier. When any character makes a deceptive Social Skill roll against a Yasuki R4+, the TN increases by 5 × School Rank. The roll engine already distinguishes Sincerity (Honesty) from Sincerity (Deceit) and the NPC Decision Engine knows whether an action’s intent is deceptive. Structurally identical to Kitsuki Investigator R2 (lying TN increase). Same implementation pattern: passive modifier keyed to opponent’s action type.

**Rank 5 — What is Yours is Mine:** If the Yasuki supplied something to the target (R3 ledger check: supplied_item_to[target_id].timestamp within last 24 IC hours), gain +5k0 to all Contested Social Rolls against that target. Conditional modifier reading the R3 ledger. NPC Decision Engine behavioral template: sequential objective chain IDENTIFY_NEED (R1c) → SOURCE_ITEM (R3) → DELIVER → PRESS_ADVANTAGE (R5).

### 29.15.3 Crab Clan — Courtier Alternate Paths

**Severed Hand [Courtier] — Replaces any Crab Bushi 4:** Strength of Bamboo: defensive trait substitution. When targeted by any contested social roll (Categories 1–4, Section 15.4), the defense Trait is replaced with Willpower regardless of what the action normally requires. Prerequisite gate: the character must have a physical Disadvantage (Missing Limb, Lame, etc.). Path eligibility is a lifecycle event — triggered when a Bushi R4+ character acquires a qualifying physical Disadvantage during play, not at creation.

**Yasuki Enforcer [Bushi] — Replaces Yasuki Courtier 2 or Hida Pragmatist 2:** Gentle Encouragement: Contested Intimidation/Willpower vs Etiquette/Willpower. On success, +1k0 attack (+1k0 per Raise) against that individual in the first Round of the next skirmish. Cross-layer modifier storage: court → combat pipeline. Entry stored as enforcer_bonus: {target_id, bonus_value, expires: next_skirmish_involving_both}. When a skirmish fires and both characters are present, bonus applies to Round 1 attack rolls, then clears. Restriction: applies to skirmish only, excludes iaijutsu duels. The cross-layer lookup pattern (social modifier persisting into combat) is reusable across other techniques.

**Yasuki Extortionist [Courtier] — Replaces Yasuki Courtier 4:** Do Me a Favor: requires the R3 supply ledger (supplied_item_to[target_id] must exist). Contested Intimidation (Control)/Willpower vs Etiquette (Courtesy)/Willpower. Two branches based on target’s financial state. Branch A (target can repay — liquid Koku ≥ item market value): inflicts Obligation, tracked as an involuntary General Favor at Moderate tier (Section 12.10). Branch B (target cannot repay): inflicts Blackmail at 3 points, entering the target’s Secret System profile as a Tier 2 secret (Section 12.9). All outputs flow through existing systems. NPC Decision Engine: prefers Branch B when available; targets characters with low Koku relative to their needs.

### 29.15.4 Crane Clan — Doji Courtier

**Rank 1 — The Soul of Honor:** Two effects. (1a) While Honor ≥ 6.0, Free Raise on all Courtier, Sincerity, and Etiquette rolls. Conditional passive modifier: roll engine checks current Honor before any of those three skill rolls. If ≥ 6.0, add one Free Raise. NPC Decision Engine implication: Doji Courtiers weight Honor preservation higher than most schools because dropping below 6.0 costs their core advantage. (1b) Contested Courtier/Awareness vs Etiquette/Awareness to discern what someone needs. Uses the DISCERN_NEED action (see 29.15.1) with Courtier/Awareness as the skill pairing. Same output as Yasuki R1c — target’s highest-priority current objective or resource shortage — but read through social grace rather than commercial instinct.

**Rank 2 — Speaking in Silence (Cadence):** Boolean flag cadence_trained on all Doji Courtier R2+ characters. When two cadence-trained characters are co-located (same court, settlement, or ASCII map zone), they may communicate silently through body language. Roll: Courtier/Awareness TN 15 for a simple instruction. Each Raise adds complexity: +1 Raise for a conditional, +2 for a multi-step plan, +3 to transfer a secret (Section 12.8). Cadence enables topic transfer (Section 15.5) with no interception risk — the communication does not enter any letter log, conversation record, or eavesdropping window. Topic awareness gained through Cadence is flagged with source: cadence, which is invisible to all detection systems. Detection by non-Crane observers: contested Investigation/Perception vs sender’s Courtier/Awareness, requires a specific reason to be watching. Cost: 0 AP (Free Action). Limitations: one-directional per use, cannot transmit precise numerical data, requires co-location. NPC Decision Engine: Crane courtier pairs at the same location automatically use Cadence each court day to synchronize objectives and share topic awareness.

**Rank 3 — The Perfect Gift:** Courtier/Awareness roll (TN 20) to identify the right gesture — object, introduction, information, invitation, or political endorsement — for a specific target. On success, writes a one-time permanent historical disposition modifier to the target: +20 (Devotion 1 equivalent per 29.15.1). Raises on the initial roll increase the modifier value (+35 at +1 Raise, +50 at +2 Raises). Non-stacking: the system checks whether a perfect_gift modifier already exists for this character pair. If yes, the technique cannot fire against that target again. One shot per relationship. Connects to R1b — a Doji who used DISCERN_NEED first has better information about what gesture to offer.

**Rank 4 — Voice of Honor:** Contested Courtier/Awareness vs Etiquette/Awareness. Fires as a replacement roll for PUBLIC_DEBATE or NEGOTIATE (Section 15.4) when the Doji R4+ invokes it. On success, the target faces a forced binary choice. Branch A (Concede): target’s topic position (Section 15.5) shifts toward the Doji’s position; small Glory loss for visibly backing down. Branch B (Refuse): target commits an etiquette breach, generating disposition penalties with witnesses and potential Honor loss. NPC Decision Engine evaluates which branch costs less: high-Makoto (Sincerity) characters lean toward Branch A; high-Ishi (Will) characters lean toward Branch B. The Doji Apologist path (29.15.5) exists specifically to repair the disposition damage from Branch A and Branch B outcomes for allies.

**Rank 5 — The Gift of the Lady:** Contested Courtier/Awareness vs Etiquette/Willpower. Fires during Direct Conversation (Category 1, 1 AP). On success, the target’s topic positions on any action topics the Doji raises are temporarily shifted to +50 (commitment threshold) for one OOC week. The shift is capped by the target’s Personality System anchors — clan loyalty, lord’s primary objective, and dominant Bushido virtue provide hard floors that cannot be overridden. During the one-week window, the target may make Public Declarations on the shifted topics; declarations are binding and irreversible regardless of how the position was achieved. After one OOC week, positions revert to their calculated baseline. No resentment or disposition penalty fires on expiry. The power of this technique lies in engineering declarations during the compliance window.

(1) Spy Network Advantage: connects to Secret System (Section 12.8). Character receives intelligence-gathering detection events at twice the normal rate — when the system runs its periodic intelligence-check cycle, the Spymaster gets checked twice instead of once. Characters who already possess the Spy Network Advantage get double frequency (effectively quadruple the base rate). (2) +2k0 to Stealth, Investigation, and Temptation: three passive skill modifiers, standard pattern. Koku drain always hits the target’s personal Koku (character scale). Roll margin = individual koku drained. Monthly cooldown timer.

Constraints: can only target disposition penalties caused by topic disagreements, court actions, or social conflicts — not deep structural modifiers like clan enmity or personal betrayal. **Doji Innocents [Courtier] — Replaces Doji Courtier 3:** The Power of Innocence: when telling the truth instead of lying for advantage, roll Sincerity (Honesty)/Awareness vs Investigation/Perception. One use per target per OOC month (already constrained by source). The NPC Decision Engine verifies the character’s statement aligns with their actual knowledge and objectives — if the statement is truthful, technique fires; if deceptive, it does not. Deliberately lying: −0.2 Honor and technique is locked out for one OOC month via timed boolean flag innocents_disabled: {expires: timestamp + 1 month OOC}.

### 29.15.6 Dragon Clan — Kitsuki Investigator

**Rank 1 — Kitsuki’s Method:** Two effects. (1a) Free Raise on all Investigation rolls: passive modifier, identical pattern to Yasuki R1a. Roll engine checks school field, adds +1 Free Raise to all Investigation rolls. (1b) Add Perception Trait Rank to Armor TN in skirmishes: passive combat defensive modifier. Perception value written into the ATN calculation when the character’s school is checked. Same pattern as other Trait-to-ATN bonuses (e.g., Shiba Bushi adding Void to ATN).

**Rank 2 — Wisdom the Wind Brings:** Anyone lying to the Kitsuki or using Feint/Disarm against the Kitsuki has their TN increased by 5 × School Rank. Two trigger contexts: social (deceptive social rolls) and combat (Feint/Disarm maneuvers). One implementation pattern covers both — passive TN modifier keyed to opponent’s action type. Structurally identical to Yasuki R4. Lying is defined per the Global Translation Rules (29.15.1): the check is on the speaker’s knowledge state and intent, not the truth value of the statement. A character innocently passing along false information they believe is not lying.

**Rank 3 — Know the Rhythm of the Heart:** Investigation (Notice)/Perception vs flat TN (target’s Intelligence × 5). On success, the Kitsuki gains the target’s full Personality System profile (Section 19) — dominant Bushido/Shourido virtues, decision-making patterns, immovable commitments. This is a stronger variant of READ_CHARACTER (Category 5, Section 15.4) with a different skill pairing and a non-contested TN. Smarter characters are harder to read. Output data already exists on every NPC sheet.

**Rank 4 — Finding the Path:** Contested Investigation (Interrogation)/Intelligence vs Etiquette/Intelligence. On success, reveals one relationship entry from the target’s Disposition System data (Section 12.2) — the highest positive or lowest negative disposition value, Kitsuki’s choice. Each Raise reveals the next entry down the list. Both sides use Intelligence — this is an intellectual contest. Cooldown: once per IC day per target. Simple timestamp check.

**Rank 5 — The Eyes Betray the Heart:** Passive triggered reaction. Whenever any character uses Sincerity (Deceit) or any action with deceptive intent against a Kitsuki R5, the system automatically fires a contested roll: Investigation/Perception vs Sincerity/Willpower. The Kitsuki does not spend AP. +5k0 bonus if the Kitsuki already holds information contradicting the statement (system checks: does the Kitsuki’s known information set contain data that contradicts the claim?). On success, the Kitsuki gains three outputs: (1) the liar’s true topic position (Section 15.5) on whatever topic they lied about; (2) the true topics the liar holds that relate to the lie — their actual known_topics entries on that subject, copied to the Kitsuki’s knowledge state; (3) the deception itself as a deployable secret (“this person lied about X”), usable through Section 12.8/12.9. The liar does not know how much the Kitsuki extracted. Every lie told to a Kitsuki R5 backfires by exposing the liar’s real information.

### 29.15.24 New Infrastructure Specifications

This subsection specifies new data structures, action definitions, and system infrastructure required by the courtier school implementations. All structures are concrete and implementation-ready. Where a technique description in 29.15.2–22 uses an ad-hoc field name and this subsection defines a standardized data structure for the same purpose, the standardized structure name and schema in this subsection is authoritative.

**New ActionID — DISCERN_NEED:** Category 5 (Intelligence). Cost: 1 AP. Skill pairing varies by school: Yasuki = Commerce/Perception, Doji = Courtier/Awareness, default = Investigation/Awareness. Defense: Etiquette/Awareness. Success: learn the target’s highest-priority current objective or resource shortage from the NPC Decision Engine’s objective stack. Failure: no information. Critical failure: target becomes aware of probing, −3 disposition. No objective tag.

**New ActionID — PROVOKE_EMOTION:** Category 4 (Lower Disposition — Public Attack). Cost: 1 AP. hostile: true. Skill: Courtier (Manipulation)/Awareness vs target’s Etiquette (Courtesy)/Willpower. Success: the target loses composure publicly. Consequences: −0.2 Honor (breach of etiquette), −0.1 Glory (public embarrassment), −3 disposition with each witness. Failure: no effect. Critical failure: the provoker’s manipulation was transparent, −5 disposition with witnesses toward the provoker. The Ikoma Bard R2a exemption: when an Ikoma Bard is the target of PROVOKE_EMOTION and the emotion is on behalf of a Lion or an honorable cause, all penalties are skipped.

**hostile Tag on ActionIDs — Complete List:** The following ActionIDs are tagged hostile: true. Category 1: INTIMIDATE. Category 3: GOSSIP, DISCLOSE, REVEAL_SECRET_PRIVATE. Category 4: PUBLIC_INSULT, PUBLIC_DEBATE, EXPOSE_SECRET_PUBLIC, PROVOKE_EMOTION. Category 5: SEARCH_PERSON. Category 6: EAVESDROP, INTERCEPT_LETTER, SEARCH_QUARTERS, FABRICATE_SECRET, FORGE_IMPERSONATION_LETTER, FORGE_ORDER, SHADOW_TARGET, SEDUCE_TO_COMPROMISE, COMMISSION_ASSASSINATION, EXECUTE_ASSASSINATION, COMMISSION_SABOTAGE. Category 7: DEMAND_TRIBUTE, CONDUCT_RAID, RAID_HARVEST, BLOCKADE_TRADE_ROUTE. Category 8: CONDUCT_STORM_ASSAULT. Category 9: IMPOSE_EMBARGO. Category 13: ISSUE_DUEL_CHALLENGE, SEEK_PRETEXT. All attack actions on the ASCII map. All offensive spell-casting targeting a specific character. Explicitly not hostile: NEGOTIATE, PERSUADE, CHARM, IMPRESS, LISTEN_REFLECT, PROBE, READ_CHARACTER, OBSERVE_COURT_ATTENDEES, BRIBE_FOR_INFO, DISCERN_NEED, SEDUCE_FOR_INFO, SEDUCE_FOR_ACCESS, SEDUCE_FOR_LEVERAGE, SEDUCE, ABORT_OPERATION, INTERVENE_CAPTAIN. COMMISSION_SABOTAGE added (was hostile-tagged in v451 but missing from this list).

**Provocation Flag:** When any character takes a hostile-tagged action against another character, the system writes a provocation entry on the victim’s record: provocation: {source_id: actor, target_id: victim, timestamp: IC day, action_id}. This flag persists indefinitely. Used by: Kasuga R2 Shell of the Tortoise (Honor penalty only fires if no provocation entry from the Kasuga exists toward the attacker), Miya Herald R1b (same pattern), and the legal system (Section 11.3) for determining who initiated a conflict. Provocation actions match the hostile ActionID list above. List is open — additions expected during development.

**Supply Ledger (Yasuki R3/R5/Extortionist):** Stored on the Yasuki character’s sheet: supply_ledger: [{target_id: int, item_id: int, item_market_value: float (Koku), timestamp: int (IC day), source_technique: string}]. Entry created when the Yasuki delivers a R3-sourced item to another character through a deliberate action (gift, trade, or favor). Cleanup: entries older than one IC season are purged. Read rules: Yasuki R5 checks target_id match within last 24 IC hours. Yasuki Extortionist checks target_id match with no time limit within the season window.

**Self-Reroll Resource (character sheet):** self_reroll: [{source: string, eligible_skills: [string], charges_current: int, charges_max: int, refresh: string, skill_swap: string or null}]. Used by: Yasuki R2 (Sincerity/Intimidation, school_rank charges, weekly, no swap), Yoritomo R3 (Sincerity, school_rank charges, weekly, swap to Intimidation Control), Kasuga R5 (School Skills, void_ring charges, weekly, no swap). The roll engine checks this array when a roll fails. If an eligible entry exists with charges remaining, the reroll fires and charges_current decrements.

**Granted Reroll (stored on ally’s character sheet):** granted_reroll: [{source_id: int, source_technique: string, bonus_dice: int, bonus_type: string (unkept/kept), uses: int, expires: int (IC day) or string, failure_penalty: {target_id: int, stat: string, value: float} or null}]. Used by: Ikoma R4 (bonus_dice = bard_honor_rank, unkept, uses: 1, expires: end_of_IC_day, failure_penalty targets bard for −0.2 Honor), Shiba Advisor (bonus_dice: null — pure reroll with no bonus, uses: 1, expires: end_of_mission, no penalty). Written by the grantor’s technique roll. Consumed when the ally fails a roll and the entry is present.

**Enhanced Void Point Spend (character sheet):** enhanced_void: [{source: string, skill: string or [string], mode: string (replace/add), bonus_fixed: string or null (e.g. “3k1”), bonus_trait: string or null, bonus_skill: string or null, kept: boolean}]. Replace mode: the enhanced bonus replaces the standard +1k1. Add mode: the bonus is added on top of +1k1. When a character spends a Void Point on a skill roll, the roll engine checks this array for a matching skill entry. Examples: Asako R1b = {skill: “Etiquette”, mode: “replace”, bonus_fixed: “3k1”}. Yoritomo Emissaries = {skill: “Etiquette (Courtesy)”, mode: “replace”, bonus_trait: “Willpower”, kept: true} and {skill: [“Etiquette”, “Sincerity”], mode: “replace”, bonus_trait: “Willpower”, kept: false}. Suzume Storytellers = {skill: “Courtier”, mode: “add”, bonus_skill: “Perform: Storytelling”}.

**Timed Advantage/Disadvantage Entries (character sheet):** timed_advantages: [{advantage_name: string, status: string (active/suppressed), source_id: int, source_technique: string, granted_timestamp: int (IC day), expires: int (IC day) or null, effects: {}}]. “Active” means the character has the Advantage and its effects apply. “Suppressed” means the character’s existing permanent Advantage is temporarily disabled. The system checks this array every IC day tick and clears expired entries. Suppressed permanent Advantages reactivate when the suppression entry expires.

**Action Block System (character sheet):** action_blocks: [{blocker_id: int, source_technique: string, blocked_action_ids: [string] or “hostile_tagged”, expires: int or string, break_conditions: [string]}]. Stored on the blocked character’s sheet. Before any action resolves, the action validation system scans this array. If the action’s ActionID matches an entry in blocked_action_ids (or the action is hostile-tagged and blocked_action_ids = “hostile_tagged”) and the action targets blocker_id, the action is denied. Break conditions are checked each tick and on relevant events. Used by: Ide R5 peace_locked (hostile_tagged, IC hours, breaks if source attacks target), Miya R3 herald_immunity (damage-dealing ActionIDs only, next_reactions_step, breaks if source takes offensive action), Otomo R3 obiesaseru (explicit list of attack/spell/movement ActionIDs, end_of_target_next_turn, no break conditions), Miya R4 blessing_ceasefire (hostile_tagged, 1 IC minute, breaks on damage received or source permission).

**Cross-Layer Modifier Storage (character sheet):** combat_modifiers_pending: [{source_id: int, source_technique: string, modifier_type: string (attack_bonus/skill_bonus), value: string (e.g. “+1k0”, “+honor_rank”), applies_to: string (e.g. “attack_rolls_round_1”, “any_one_skill_roll”), target_id: int or null, uses: int or “unlimited”, expires: string (“end_of_battle”, “end_of_round_1”, “end_of_IC_day”)}]. When a skirmish or Mass Battle fires, the combat system reads this array for all participating characters. Matching entries are activated and applied. After use or expiry, entries are cleared. Modifiers only activate when combat actually happens — if no combat occurs before expiry, they fade unused. Used by: Yasuki Enforcer (attack_bonus, +1k0 per Raise, attack_rolls_round_1, specific target_id, skirmish only excludes duels) and Ikoma Bard R3 (skill_bonus, +honor_rank, any_one_skill_roll, null target, end_of_battle).

**Third-Party Disposition Write (Otomo Seiyaku):** When the Otomo provokes disagreement, the modifier is written to both victims’ disposition records toward each other. Category: temporary (decays at standard rate per Section 12.2). Structure: {category: “temporary”, value: −15, source_type: “seiyaku_provocation”, source_actor: null (hidden — not traced to Otomo), apparent_source: “topic_disagreement”, timestamp: int (IC day), decay_rate: standard}. Detection requires an Investigation action that could trace the friction back to the Otomo.

**Displayed Art Field (location record):** displayed_art: {item_id: int, creator_id: int, placed_timestamp: int (IC day), exclusion_list: [int]} or null. Supported on: Castle/Shiro, court locations, Temple/Shinden, estate/private quarters. Not supported on: open wilderness, military camps, roads, zones without permanent structures. One slot per location, no stacking. Full Lesser Zone subtype definitions are in Section 57.36 (Flag Matrix). The displayed_art_slot flag is defined per zone subtype: true for OHIROMA, ENKAI_HALL, AUDIENCE_CHAMBER, LORD_QUARTERS, TEMPLE_GROUNDS, and PLEASURE_QUARTER. False for all wilderness, functional, and exterior zones.

**Zone Event Log (per Lesser Zone):** zone_event_log: [{event_type: string (combat/covert_action/social_interaction/topic_generated/character_visit/death), participants: [int] (character IDs), action_id: string or null, topic_id: int or null, timestamp: int (IC day), detail_level: {category: string, outcome: string or null}}]. Write trigger: any time a named character performs an action at this zone, an entry is written automatically. Cleanup: entries older than one IC season are purged. Read by: Kitsuki’s Eye (filtered by Raise count — base success = category only, +1 Raise = one known participant, +2 = event plus one participant, +3 = full fragment, +4 = multiple fragments and timeline), magistrate investigations (Section 11.3, pending), Secret System detection checks, insurgency detection. The Eye’s TN increases with age within the season: current IC month = base TN 25, previous IC month = TN 30, older = TN 35. General-purpose location history infrastructure serving multiple systems.

**Character Awareness Tracking (met_characters):** met_characters: [int] set on each character’s sheet. When two named characters are co-located in the same zone with an ASCII map (Lesser Zone or Navigation Zone per Section 4.3.4 v564 amendment), both add the other’s ID to their set. This is a global constraint: a character can only reference another character in conversation, topics, letters, or court actions if (1) the other character’s ID is in their met_characters set (met in person), or (2) the other character is named in a topic in their known_topics, referenced in a letter they received, or identified through an intelligence action. A character who has neither met nor heard of someone cannot target them with any social action. The Kitsuki’s Eye can only identify zone event log participants whose IDs are in the Eye’s met_characters set. This rule applies globally across all systems — placement in Section 22 or Section 55 pending.

**Automated Intelligence Query Scoring (Otomo R2, Kasuga R4):** When these techniques fire successfully, the system scores every undiscovered intelligence item in the relevant pool. Scoring: severity (Tier 1 = 40, Tier 2 = 30, Tier 3 = 20, Tier 4 = 10) + objective relevance (relates to character’s primary objective = +15, otherwise 0) + proximity (subject is present at character’s current location = +10, otherwise 0) + recency (current IC month = +10, previous IC month = +5, older = 0). Hard filter: items already in the character’s knowledge pool are excluded. Highest score wins. Otomo pool: secrets and topics about characters at the current court, plus topics flowing through Imperial-connected information networks. Kasuga pool: secrets and topics related to criminal activity, insurgencies, smuggling, corruption, and underworld figures at the current settlement.

### 29.15.25 Concept Definitions

**Disadvantage-to-Secret Severity Mapping (Bayushi R3):** When a revealed Disadvantage becomes a secret topic object, severity tier is assigned by type. Only concealable Disadvantages are mapped — physical disabilities (Blind, Lame, Missing Limb), visible conditions (Bad Eyesight, Disturbing Countenance, Epilepsy), and public statuses (Infamous, Forced Retirement) cannot be hidden and are not valid secret topics. Tier 1 (existential): Dark Secret, Shadowlands Taint, Sleeper Agent. Also applies to character states: Maho Corruption, Kolat membership, Lost status. Tier 2 (severe): Compulsion (Opium/Gambling), Black Sheep, Consumed (obsessive), Failure of Bushido (any virtue), Dishonored, Cast Out (3pt — severed from family), Momoku (spiritual blindness), Blackmailed (reveals both the secret and the leverage chain). Tier 3 (significant): Lechery, Greedy, Jealousy, Cowardice (Failure of Bushido: Courage, if not already revealed), Overconfident, Sworn Enemy (with an ally of the discoverer), Phobia (exploitable fear), Debt (financial compromise), Obligation (political compromise), Rumormonger (reveals character as untrustworthy), True Love (loved one becomes a target), Doubt (exploitable self-doubt), Gullible (exploitable naivety). Tier 4 (minor): Brash, Contrary, Idealistic, Soft-Hearted, Lost Love, Small, Ascetic (cultural eccentricity), Insensitive, Obtuse, Gaijin Name (embarrassing heritage), Antisocial, Bitter Betrothal (personal unhappiness), Can’t Lie (minor — arguably honorable). **Paragon of Bushido / Failure of Bushido (Ikoma Orator):** Verbatim from L5R 4e Core Rulebook. Paragon (7 points, Mental): Compassion — Void Point on roll to help those of lower Status grants +2k2 instead of +1k1. Courage — +1k1 to resist Intimidation or overcome Fear. Courtesy — +2k0 on Etiquette to avoid embarrassment or breach. Duty — spend Void Point on any Skill/Spell Roll to negate all TN penalties including Wounds. Honesty — +1k1 on Sincerity (Honesty) rolls. Honor — add twice Honor Rank to resist Temptation/Intimidation. Sincerity — +2k0 on all Contested Sincerity rolls. Global bonus: +0.1 Honor any time the character gains Honor for exhibiting the Paragon virtue. Failure of Bushido (varies, Mental): Compassion (3 pts) — must spend Void Point to act on behalf of lower Status. Courage (4 pts) — +5 TN against higher Glory/Status or Shadowlands opponents. Courtesy (4 pts) — must call a wasted Raise on Social Rolls to apologize or avoid offense. Duty (6 pts) — cannot spend Void to negate Wounds. Honesty (3 pts) — cannot spend Void on Sincerity (Honesty) rolls. Honor (3 pts) — cannot add Honor Rank to resist Intimidation/Temptation. Sincerity (4 pts) — must call a wasted Raise on Sincerity rolls to convince. The Ikoma Orator grants a Paragon (to an ally) or inflicts a Failure (on an enemy) as a timed entry lasting one OOC week using the timed_advantages infrastructure.

**Sacrosanct Advantage:** Already defined in the Advantage reference. Requires Honor 6.0+. As long as Honor does not drop below 6.0, no character with Honor Rank 5+ may attack you unless you attack first. Duels are exempt. The Imperial War Observer gains this as a timed entry active during war reporting duties.

**Blocked Objective Handling (Otomo Bureaucrat):** When Imperial Scrutiny succeeds, the target’s primary objective is flagged: blocked: {expires: timestamp, source_id: bureaucrat_id}. The engine skips this objective during action selection — any action scored against it gets score 0. The engine evaluates secondary objectives and selects the highest-scoring one as the acting objective for the duration. If no secondary objectives exist, the character defaults to passive actions (CHARM, social maintenance, DO_NOTHING). When the block expires, the primary objective is re-evaluated normally on the next AP tick.

**Pull the String Deadline Mapping (Scorpion Instigator R3):** Deadline scales with action type. Court actions (Categories 1–5): end of current court day. Communication (Category 10): end of current IC day. Economic actions (Category 9): end of current IC season. Military actions (Category 7): end of current IC season. Travel-dependent actions: geographic travel time (Section 4.4) plus one IC season buffer. Administrative actions (Category 14): end of current IC season. Covert actions (Category 6): end of current IC season. Honor and Dueling (Category 13): end of current IC day for the challenge/atonement; duel resolution follows the pending duel preparation window. Kolat Operations (Category 16): cannot be compelled. If the suggested action does not map to an existing ActionID, it is rejected.
