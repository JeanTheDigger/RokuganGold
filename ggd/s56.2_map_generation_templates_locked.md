## 56.2 Map Generation Templates --- LOCKED

Each ASCII map mission is built from a procedurally generated layout. Templates are not premade maps. They are generation rulesets --- algorithms that build a different map every time using different rules governing the output. The occupants and dressing change per quest seed; the spatial generation logic is reusable across multiple seed types.

Seven base templates are defined. Each template specifies its own rules for space carving, entry and flow, room/area types, population placement, environmental features, and objective placement.

### 56.2.1 Template Probability Pools by Province Terrain --- LOCKED

When a quest seed fires, the province terrain type determines which map templates are available and at what probability. The generator rolls against the pool and produces the selected template. If the insurgency Strength exceeds the template’s maximum housing capacity, the roll is discarded and rerolled against templates that can accommodate the population.

**Plains / Flatlands:** Occupied Village 45%, Makeshift Stockade 35%, Ruined Structure 15%, Ravine Camp 5%.

**River Delta / Coastal Lowlands:** Occupied Village 50%, Makeshift Stockade 20%, Ruined Structure 20%, Ravine Camp 10%.

**Forest / Light Hills:** Forest Approach + Camp 40%, Cave 20%, Hilltop Position 15%, Occupied Village 15%, Ruined Structure 10%.

**Heavy Hills / Rough Terrain:** Cave 30%, Hilltop Position 25%, Ravine Camp 20%, Ruined Structure 15%, Forest Approach + Camp 10%.

**Mountains:** Cave 50%, Ravine Camp 20%, Hilltop Position 20%, Ruined Structure 10%.

**Ruined Structure Conditional Availability --- LOCKED:** The Ruined Structure template is only available if an actual ruin exists in the province. The generator checks province history before rolling --- war damage, famine abandonment, Taint corruption, revolt destruction, or natural decay must have produced a specific ruined building (per Section 56.12.1). If no ruin exists, the Ruined Structure percentage redistributes proportionally to the remaining templates. A province with a rich history of conflict has more template variety than a peaceful one. If multiple ruins exist, the generator selects a specific ruin --- its origin determines the architecture type and condition. The bandits moved into a particular abandoned building, not an abstract concept.
