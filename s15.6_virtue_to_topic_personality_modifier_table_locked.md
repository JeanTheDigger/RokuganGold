## 15.6 Virtue-to-Topic Personality Modifier Table — LOCKED

This table defines the flat modifier each virtue applies to a character's starting position on a topic (Input 2 of the Starting Position Formula, Section 15.5). Modifiers fire on topic type regardless of tier — a Shadowlands Incursion and a Provincial Famine both trigger Jin's suffering modifier. Only genuine instinctive leans are listed — all unlisted virtue×topic pairings are 0.

IMPLEMENTATION NOTE: Several topic types require variant flags to determine which modifier fires. A Death topic needs a variant field (honorable / suspicious-or-violent / battlefield). A Battle Outcome needs a variant (victory / defeat / atrocity). A Disgrace needs a variant (unjust / justified). An Alliance needs a variant (formed / broken). These flags are part of the topic data structure and must be set when a topic is generated. See Section 15.7 for the full topic variant flag specification.

**Border Conflict — Note:**

Border Conflict uses the same virtue modifiers as Provincial Raid. It is the same type of event at a larger scale: Yu +6, Ketsui +5, Kyōryōku +8. No separate entry required.

**Siege Ending — Variant Flag:**

Siege Ending requires a variant flag: siege_end_variant. Values: successful_defense (garrison held), negotiated_peace (terms agreed), surrender_forced (garrison capitulated), relief_army (external force broke the siege). Add siege_end_variant to the Section 15.7 variant flag specification.

**Adoption — Variant Flag:**

Adoption requires a variant flag: adoption_variant. Values: clear_succession (legitimate heir adopted with broad acceptance), questionable_legitimacy (adoption contested or circumstances suspicious). Add adoption_variant to Section 15.7.
