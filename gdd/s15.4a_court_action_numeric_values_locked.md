## 15.4a Court Action Numeric Values — LOCKED

This section formalizes numeric values for the five contested Direct Conversation actions
from Section 15.4. The main section specifies outcomes qualitatively ("Moderate disposition
gain", "Strong disposition gain", "slightly lower TN") without numeric values. This addendum
assigns the canonical values used by the simulation.

---

### Source Derivation

**GDD-confirmed (from Public Debate text in s15.4):**

The Public Debate section states: *"Broadcast position movement is deliberately smaller per
person than targeted actions (Negotiate: +8, Persuade: +12) because debate touches everyone
lightly rather than moving one person deeply."*

This sentence defines the base-success position shift for Negotiate (+8) and Persuade (+12)
as direct specifications embedded in the comparison. These are authoritative GDD values.

**Calibration anchors for disposition gain and loss values:**

| Source | Value |
|--------|-------|
| Play a Game (pleasant recreation, both directions) | +3 |
| Gossip base damage (targeted, 1 AP) | −5 |
| Public Debate "slight" per witness (score 1–2) | ±1 |
| Public Debate "moderate" per witness (score 3–4) | ±2 |
| Public Debate "strong" per witness (score 5–6) | ±3 |
| Public Debate "decisive" per witness (score 7+) | ±4 |

Targeted one-on-one actions affect a single person more deeply than per-witness broadcast
effects. "Moderate" for a targeted action means stronger than Play a Game and strong enough
to meaningfully advance within a disposition tier. "Strong" means stronger than "Moderate".
Charm is intentionally weaker per action than Negotiate to reflect its shallowness and
ceiling limitation.

### Negotiate — LOCKED

Awareness + Courtier vs Awareness + Courtier.

| Outcome | Disposition | Position shift |
|---------|-------------|----------------|
| Success base | +6 | +8 |
| Each Raise (disposition) | +2 | — |
| Each Raise (position) | — | +4 |
| Future Negotiate TN reduction this session | — | −5 |
| Failure | 0 | hardens (−1) |
| Critical failure (margin ≤ −10) | −3 | hardens significantly (−3) |

Position hardens on any failure. The failure position shifts are derived from the Public
Debate per-witness scale: "hardens slightly" = −1 ("slight" tier = ±1); "hardens
significantly" = −3 ("strong" tier = ±3). Raise disposition and position bonuses stack
additively.

### Persuade — LOCKED

Awareness + Sincerity vs Willpower + Sincerity.

| Outcome | Disposition | Position shift |
|---------|-------------|----------------|
| Success base | +9 | +12 |
| Each Raise (disposition) | +3 | — |
| Each Raise (position) | — | +5 |
| Failure | 0 | 0 |
| Critical failure (margin ≤ −10) | −5 | hardens |

Position moved by Persuade is **durable** (marked `position_durable`). No TN reduction
granted — Persuade is high-impact but doesn't lower future barriers. The critical failure
disposition loss is −5 (Gossip anchor, see Design Rationale below) because s15.4 does
not qualify Persuade's critical failure as "small" — it is a genuine backfire, not a
minor social stumble.

### Design Rationale

**"Moderate" = +6 targeted.** Two Moderate successes (no Raises) advance from zero to +12,
approaching the Acquaintance threshold (+11). This calibration ensures targeted actions feel
meaningful without trivially crossing tier boundaries in a single exchange.

**"Strong" = +9 targeted.** Stronger than Moderate as s15.4 requires. A successful
Persuade plus a successful Negotiate (both no Raises) reaches +15, solidly into
Acquaintance and beginning to push toward Friend (+31 threshold). Requires multiple sessions
of focused investment to reach the deepest trust tiers.

**Charm at +5.** Intentionally weaker than Negotiate (+6) despite both costing 1 AP. Charm
changes feelings without touching positions, so it trades lower depth for lower risk. The
ceiling at +40 enforces the s15.4 design intent that Charm alone cannot produce deep bonds.

**Raise bonuses scale with tier: +2 for Moderate-tier (Charm, Negotiate, Impress), +3 for
Strong-tier (Persuade, Listen/Reflect).** Consistent with Public Debate's +2/step Raise
increment, scaled for targeted depth.

**TN reductions: −5 for "slight" (Negotiate, Impress), −10 for Listen/Reflect.** The
distinction reflects s15.4's language: Negotiate and Impress grant "slight" reductions,
while Listen/Reflect explicitly "opens them to future Persuade or Negotiate" — a stronger
enabling effect. −10 is equivalent to one full Raise advantage on the follow-up roll.

**Position raise bonuses: +4 for Negotiate, +5 for Persuade.** Derived from the Public
Debate pattern (+2 per step at broadcast scale), scaled up for targeted single-person depth.
Persuade's bonus is slightly higher to reflect its "more significantly and durably" character
per s15.4.

**Critical failure penalties: "small" = −3; Persuade (unqualified) = −5.**
s15.4 describes critical failures for Charm, Negotiate, Impress, and Listen/Reflect as
"small disposition loss." Persuade's critical failure is described simply as "disposition
loss" — no "small" qualifier — matching s15.4's framing of Persuade as the highest-risk,
highest-reward action.

"Small" = −3 is calibrated against Play a Game (±3), which the GDD establishes as the
magnitude of pleasant recreation. A "small" social stumble mirrors this in the negative
direction and sits clearly below Gossip base damage (−5), which is the canonical targeted
social attack. Values larger than −5 cannot be "small" per this calibration.

Persuade = −5 uses Gossip as the anchor: Gossip is explicitly designed to damage a
relationship at targeted-action magnitude, and a Persuade that appeals to something the
target finds offensive achieves the same result — a targeted, relationship-damaging moment.

Negotiate failure position shifts: −1 ("hardens slightly") and −3 ("hardens significantly")
are derived from the Public Debate per-witness scale directly. "Slight" = ±1 and "strong"
(≈ "significantly") = ±3 on that scale.
