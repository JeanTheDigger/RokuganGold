## Locked Mechanics

### Noshi (s57.26.6–57.26.8)

CRAFT action: Artisan: Origami / Awareness vs TN 15. AP cost 1.
Quality by Raises: 0 = Normal, 1 = Fine, 2 = Exceptional, 3 = Masterwork, 4+ = Legendary.
Failure → Mundane (noshi_quality_tier = 0; stored on item with flag).

On DELIVER_GIFT: noshi Free Raises by tier:
- NORMAL (1) → +0 FR
- FINE (2) → +1 FR
- EXCEPTIONAL (3) → +2 FR
- MASTERWORK (4) → +3 FR
- LEGENDARY (5) → +4 FR

Mundane penalty: -1k0 on gift roll (noshi_is_mundane flag set on item).
Noshi is consumed on DELIVER_GIFT. Self-wrapping exclusion: if wrapper_id == item_owner_id,
no wrapper disposition bonus fires.

Wrapper disposition bonus (A1, locked per s57.26.8):
By quality tier on CRAFT completion when wrapping for another:
- NORMAL (1) → +2, 30 IC days
- FINE (2) → +4, 45 IC days
- EXCEPTIONAL (3) → +6, 60 IC days
- MASTERWORK (4) → +8, 75 IC days
- LEGENDARY (5) → +10, 90 IC days

Stored as historical modifier category "noshi_wrapper_bonus" on gift-giver toward wrapper.

### Gohei (s57.26.12–57.26.13)

CRAFT action: Artisan: Origami / Awareness vs TN 20. AP cost 1.
Quality by Raises: 0 = Normal, 1 = Fine, 2 = Exceptional, 3 = Masterwork, 4+ = Legendary.
Stored as inventory item (item_type: "gohei").

Uses_remaining by quality (A2, locked per s57.26.12):
- NORMAL (1) → 3
- FINE (2) → 5
- EXCEPTIONAL (3) → 8
- MASTERWORK (4) → 12
- LEGENDARY (5) → 20

Free Raises on PERFORM_WORSHIP (A3, locked per s57.26.13):
- NORMAL (1) → +0 FR
- FINE (2) → +1 FR
- EXCEPTIONAL (3) → +2 FR
- MASTERWORK (4) → +3 FR
- LEGENDARY (5) → +4 FR

One gohei applies per roll (highest quality). Uses_remaining ticks down by 1 per worship use.
Destroyed when uses_remaining reaches 0. Gohei may be gifted (uses_remaining carries over).

### Senbazuru (s57.26.14–57.26.17)

Declaration: 0 AP. Creates SenbazuruData record. One active senbazuru at a time (gate clears on presentation, not completion). Generates Tier 4 topic.

Dedication types: Healing (named sick/wounded/Tainted recipient), Protection (named recipient facing danger), Remembrance (recently deceased recipient — within last IC season), Atonement (no recipient — self).

Progress: CRAFT action, Artisan: Origami / Awareness vs TN 15. AP cost 1.
Success: base 10 cranes + 5 per declared Raise. Raises added to total_raises, successful_session_count += 1.
Failure: 0 cranes, AP spent, session lost.

Crane counts per Raises (A4, locked per s57.26.15):
- 0 Raises (TN 15) → 10 cranes
- 1 Raise (TN 20) → 15 cranes
- 2 Raises (TN 25) → 20 cranes
- 3 Raises (TN 30) → 25 cranes
- 4 Raises (TN 35) → 30 cranes

Quality at 1000 cranes: avg_raises = total_raises / successful_session_count (round down).
- avg 0 → NORMAL, avg 1 → FINE, avg 2 → EXCEPTIONAL, avg 3 → MASTERWORK, avg 4+ → LEGENDARY

Completion: is_complete = true, completion_date set. Generates topic (TIER_3 for Exceptional+, TIER_4 for Fine/Normal).

Presentation (1 AP action, is_complete must be true):

**Healing:**
First effect — recipient disposition toward folder: NORMAL=+3, FINE=+5, EXCEPTIONAL=+8, MASTERWORK=+12, LEGENDARY=+15 (Category 3 durations: 30/45/60/75/90 IC days).
Second effect (only if recipient still wounded/ill/Tainted) — Free Raises on next qualifying roll: NORMAL=+0, FINE=+1, EXCEPTIONAL=+2, MASTERWORK=+3, LEGENDARY=+4.

**Protection:**
First effect — disposition: same scaling as Healing.
Second effect — recovers all spent Void Points. If already at full Void, +1k0 on first dangerous roll.

**Remembrance:**
Effect 1 — all present witnesses gain disposition toward folder: NORMAL=+2, FINE=+4, EXCEPTIONAL=+6, MASTERWORK=+8, LEGENDARY=+10 (Category 3 durations).
Effect 2 — folder gains Glory: NORMAL=+0.1, FINE=+0.2, EXCEPTIONAL=+0.3, MASTERWORK=+0.4, LEGENDARY=+0.5.

**Atonement:**
Honor recovery: NORMAL=+0.05, FINE=+0.1, EXCEPTIONAL=+0.15, MASTERWORK=+0.2, LEGENDARY=+0.3.

Post-presentation: presentation_date set, state = "presented". Senbazuru record retained as history (no longer blocks new declarations).

**Dedication shift:** If Healing/Protection recipient dies while project is in progress, dedication_type shifts to "Remembrance" automatically. Progress unaffected. Tier 4 topic.

**Creator death:** state → "creator_deceased". Tier 4 topic.
