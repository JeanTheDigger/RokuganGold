## 4.7 Multiplayer Structure

The game is designed for multiple players existing in the same living world simultaneously, each controlling their own character.

- Players share the same world simulation — political changes affect all players’ situations.

- Players can occupy the same ASCII map and interact directly (ally, fight, talk).

- Players from different clans have inherently different political interests, enabling organic conflict and alliance.

- Player-vs-Player combat uses the same L5R4e resolution system as PvE.

✓ DONE — Persistent public server. Players drop in and out of a living world. Designed to work with as few as 5–10 players (clan AI fills all unoccupied roles) but scales to unlimited. See Section 13 for full time system, Bubble Time RP rules, and offline character behavior.

✓ DONE — Server-authoritative. The server runs the world simulation continuously (resource ticks, clan AI, seasonal events) regardless of who is online. See Section 13.

✓ DONE — Offline characters persist in their holdings in a default resting state, maintained by standing orders issued to their household. They continue to receive stipends, consume Rice, and are subject to world events, but cannot take active actions. See Section 13.
