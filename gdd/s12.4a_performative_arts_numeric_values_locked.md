## 12.4a Performative Arts Numeric Values — LOCKED

This section formalizes two numeric values left unspecified by GDD s12.4 and s15.4.
The main sections describe PERFORM_FOR outcomes qualitatively ("strong disposition gain",
"small disposition loss") without assigning numbers.

---

### Source Derivation

**GDD-confirmed anchors (from s15.4a and s12.4):**

| Anchor | Value | Source |
|--------|-------|--------|
| Charm base disposition (shallow, ceiling-limited) | +5 | s15.4a |
| Negotiate / Impress base (moderate) | +6 | s15.4a |
| Persuade / Listen+Reflect base (strong) | +9 | s15.4a |
| "Small disposition loss" (gift critical failure, Charm critical) | −3 | s12.3a, s15.4a |
| PUBLIC_PERFORMANCE success per witness | +2 | s12.4 (GDD-confirmed) |

**PERFORM_FOR vs PUBLIC_PERFORMANCE:**

PERFORM_FOR is targeted (one recipient). PUBLIC_PERFORMANCE is broadcast (all witnesses,
weaker per person). The same principle governs this distinction as Persuade vs Public Debate:
targeted one-on-one actions move one person more deeply than broadcast effects.

PERFORM_FOR raises add +1 per raise (matching PUBLIC_PERFORMANCE's per-witness raise bonus).
This lower raise multiplier (vs Persuade's +3/raise) reflects that an artistic gesture,
however masterful, has lower ceiling than a direct persuasion technique.

### Values — LOCKED

| Constant | Value | Rationale |
|----------|-------|-----------|
| `PERFORM_FOR_SUCCESS_DISPOSITION` | +8 | GDD: "strong disposition gain." Clearly above moderate actions (Negotiate/Impress: +6). Approaches Persuade base (+9) as artistic gesture. Regular success = +8; masterful (2 raises) = +10 + 0.2 Glory. |
| `PERFORM_FOR_FAILURE_DISPOSITION` | −3 | GDD: "small disposition loss." Consistent with all "small" losses across the system (gift critical failure, Charm critical failure). One person affected. |

### Notes

The PERFORM_FOR failure path covers all failure outcomes (regular and critical failure
are not distinguished for PERFORM_FOR by the GDD — only success and failure are specified).

PUBLIC_PERFORMANCE constants are GDD-confirmed and unchanged:
- `SUCCESS_DISPOSITION` = +2 per witness
- `CRITICAL_FAILURE_DISPOSITION` = −2 per witness
- `SUCCESS_GLORY` = 0.3
- `MASTERFUL_GLORY` = 0.2 (used for PERFORM_FOR masterful only per GDD s12.4)
