# 29. School Reference Data

This section documents all school data for the Great Clans. Each entry includes school type, benefit, starting skills, honor, outfit, and technique summaries. Alternate Paths replace a specific rank of a Basic School and provide a single technique. Advanced Schools have entry requirements. Source: L5R 4th Edition Core Rulebook [CR], Great Clans [GC], Emerald Empire [EE], Imperial Histories [IH], and various Book of supplements.
